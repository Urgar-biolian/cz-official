# 蓝桥杯板块接口文档

## 概述

本文档描述了蓝桥杯板块相关的API接口，包括获奖者管理、比赛信息等功能。

## 基础信息

- **基础URL**: `http://localhost:3000/api`
- **数据格式**: JSON
- **认证方式**: JWT Token (部分接口需要)

## 数据模型

### Winner (获奖者)

```typescript
interface Winner {
  id: number;           // 获奖者ID
  name: string;         // 获奖者姓名
  competition: string;  // 比赛名称
  award: string;        // 获奖等级 (如: "国一", "国二", "国三")
  avatar: string;       // 头像URL
  createdAt: Date;      // 创建时间
}
```

## API 接口

### 1. 获取所有获奖者

**接口地址**: `GET /api/winners`

**接口描述**: 获取所有蓝桥杯获奖者信息，按创建时间倒序排列

**请求参数**: 无

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "张三",
      "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
      "award": "国一",
      "avatar": "avatar1.jpg",
      "createdAt": "2024-01-15T10:30:00.000Z"
    },
    {
      "id": 2,
      "name": "李四",
      "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
      "award": "国二",
      "avatar": "avatar2.jpg",
      "createdAt": "2024-01-14T15:20:00.000Z"
    }
  ]
}
```

**响应字段说明**:
- `code`: 状态码，200表示成功
- `message`: 响应消息
- `data`: 获奖者列表数据

### 2. 获取获奖者详情

**接口地址**: `GET /api/winners/:id`

**接口描述**: 根据ID获取特定获奖者的详细信息

**路径参数**:
- `id` (number): 获奖者ID

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "name": "张三",
    "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
    "award": "国一",
    "avatar": "avatar1.jpg",
    "createdAt": "2024-01-15T10:30:00.000Z"
  }
}
```

### 3. 创建获奖者

**接口地址**: `POST /api/winners`

**接口描述**: 创建新的获奖者记录 (需要管理员权限)

**请求头**:
```
Authorization: Bearer <jwt_token>
Content-Type: application/json
```

**请求体**:
```json
{
  "name": "王五",
  "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
  "award": "国三",
  "avatar": "avatar3.jpg"
}
```

**请求字段说明**:
- `name` (string, 必填): 获奖者姓名
- `competition` (string, 必填): 比赛名称
- `award` (string, 必填): 获奖等级
- `avatar` (string, 可选): 头像文件名

**响应示例**:
```json
{
  "code": 201,
  "message": "获奖者创建成功",
  "data": {
    "id": 3,
    "name": "王五",
    "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
    "award": "国三",
    "avatar": "avatar3.jpg",
    "createdAt": "2024-01-16T09:15:00.000Z"
  }
}
```

### 4. 更新获奖者信息

  "data": {
    "id": 3,
    "name": "王五",

### 5. 删除获奖者

**接口地址**: `DELETE /api/winners/:id`

**接口描述**: 删除指定的获奖者记录 (需要管理员权限)

**请求头**:
```
Authorization: Bearer <jwt_token>
```

**路径参数**:
- `id` (number): 获奖者ID

**响应示例**:
```json
{
  "code": 200,
  "message": "获奖者删除成功",
  "data": null
}
```

**查询参数**:

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误 |
| 401 | 未授权，需要登录 |
| 403 | 权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 错误响应格式

```json
{
  "code": 400,
  "message": "请求参数错误",
  "error": "name字段不能为空"
}
```

## 认证说明

需要认证的接口需要在请求头中包含JWT Token：

```
Authorization: Bearer <your_jwt_token>
```

获取Token的方式请参考认证相关接口文档。

## 文件上传说明

头像文件上传使用现有的文件上传接口：

**接口地址**: `POST /api/upload`

**请求头**:
```
Authorization: Bearer <jwt_token>
Content-Type: multipart/form-data
```

**请求体**:
- `file`: 图片文件

**响应示例**:
```json
{
  "code": 200,
  "message": "文件上传成功",
  "data": {
    "filename": "avatar_123456.jpg",
    "url": "/uploads/avatar_123456.jpg"
  }
}
```

## 前端集成示例

### Vue 3 组件使用示例

```vue
<template>
  <div class="winner-list">
    <div v-for="winner in winners" :key="winner.id" class="winner-item">
      <img :src="winner.avatar" :alt="winner.name">
      <h3>{{ winner.name }}</h3>
      <p>{{ winner.competition }}</p>
      <span class="award">{{ winner.award }}</span>
    </div>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import axios from 'axios'

const winners = ref([])

const fetchWinners = async () => {
  try {
    const response = await axios.get('/api/winners')
    winners.value = response.data.data
  } catch (error) {
    console.error('获取获奖者列表失败:', error)
  }
}

onMounted(() => {
  fetchWinners()
})
</script>
```

## 注意事项

1. 所有时间字段使用ISO 8601格式
2. 头像文件建议使用JPG或PNG格式，大小不超过2MB
3. 获奖等级建议使用标准格式：国一、国二、国三、省一、省二等
4. 比赛名称建议使用完整名称
5. 需要管理员权限的接口请确保用户具有相应权限
