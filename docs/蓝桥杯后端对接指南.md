# 蓝桥杯后端对接指南

本指南面向**其他前端项目**，介绍如何对接本项目的蓝桥杯后端 API，包括接口地址、认证方式、请求示例、注意事项等。

---

## 1. 后端基础信息

- **后端基础地址**（Base URL）：
  - 本地开发示例：`http://localhost:3000/api`
  - 局域网/服务器示例：`http://localhost:3000/api`
- **接口文档**：详见 [`docs/蓝桥杯接口文档.md`](./蓝桥杯接口文档.md)
- **数据格式**：所有接口均为 JSON

---

## 2. 认证与权限

- **部分接口（如新增/编辑/删除）需要 JWT Token 认证**
- 登录后端获取 token，后续请求需在 header 中携带：
  ```
  Authorization: Bearer <你的token>
  ```
- 普通查询接口（如获奖者列表）无需认证

---

## 3. 典型API接口

### 3.1 获取所有获奖者

- **接口地址**：`GET /api/winners`
- **请求参数**：无
- **响应结构**：
  ```json
  {
    "code": 200,
    "message": "success",
    "data": [
      {
        "id": 1,
        "name": "张三",
        "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
        "award": "国一",
        "avatar": "avatar1.jpg",
        "createdAt": "2024-01-15T10:30:00.000Z"
      }
      // ...
    ]
  }
  ```

### 3.2 创建获奖者（需管理员 Token）

- **接口地址**：`POST /api/winners`
- **请求头**：
  ```
  Authorization: Bearer <你的token>
  Content-Type: application/json
  ```
- **请求体**：
  ```json
  {
    "name": "张三",
    "competition": "蓝桥杯全国软件和信息技术专业人才大赛",
    "award": "国一",
    "avatar": "avatar1.jpg"
  }
  ```
- **响应结构**：
  ```json
  {
    "code": 201,
    "message": "获奖者创建成功",
    "data": { ... }
  }
  ```

### 3.3 更新/删除获奖者（需管理员 Token）

- `PUT /api/winners/:id`
- `DELETE /api/winners/:id`
- 需携带管理员 Token，参数同上。

### 3.4 按奖项/比赛筛选

- `GET /api/winners/filter/award?award=国一`
- `GET /api/winners/filter/competition?competition=蓝桥杯全国软件和信息技术专业人才大赛`

---

## 4. 用户认证

- **登录接口**：`POST /api/login`
  - 请求体：`{ "email": "xxx", "password": "xxx" }`
  - 响应：包含 `token` 字段
- **注册接口**：`POST /api/register`
- **Token 用法**：管理员操作（新增/编辑/删除）需在 header 加 `Authorization: Bearer <token>`

---

## 5. 文件上传

- **上传图片**：`POST /api/upload/image`
  - 表单字段名：`file`
  - 响应：`{ "url": "http://<服务器>/uploads/xxx.jpg" }`
- **注意**：仅支持图片类型，大小限制2MB

---

## 6. 返回结构与错误码

- **所有接口返回结构**：
  ```json
  {
    "code": 200,
    "message": "success",
    "data": { ... }
  }
  ```
- **错误示例**：
  ```json
  {
    "code": 401,
    "message": "未授权",
    "error": "Unauthorized"
  }
  ```

---

## 7. 跨域（CORS）与代理

- **后端已全局开启 CORS**，外部前端可直接请求。
- 如遇跨域问题，检查前端代理或联系后端。

---

## 8. 常见问题

- **404 Not Found**：检查接口路径是否为 `/api/xxx`，后端服务是否已启动
- **401 Unauthorized**：需要登录并携带有效的 JWT Token
- **403 Forbidden**：账号权限不足（如非管理员）
- **500 Internal Server Error**：检查请求参数是否完整、格式是否正确

---

## 9. 推荐 axios 封装示例

```js
import axios from 'axios'

const api = axios.create({
  baseURL: 'http://<你的服务器IP>:3000/api',
  timeout: 10000
})

// 获取获奖者列表
api.get('/winners').then(res => console.log(res.data))

// 创建获奖者（需 token）
api.post('/winners', {
  name: '张三',
  competition: '蓝桥杯全国软件和信息技术专业人才大赛',
  award: '国一',
  avatar: 'avatar1.jpg'
}, {
  headers: {
    Authorization: 'Bearer <你的token>'
  }
})
```

---

## 10. 其它说明

- 所有时间字段为 ISO 字符串
- 头像字段为图片文件名或完整 URL
- 需要上传头像请使用 `/api/upload/image` 接口
- 如需批量导入/导出，请联系后端开发
